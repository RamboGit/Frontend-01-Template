# 工具链第一课（项目初始化阶段）

## 目前所用工具：
### npm [npm](https://docs.npmjs.com/)
 npm 由三个独立的部分组成：
    * 网站： 开发者查找包（package）、设置参数以及管理 npm 使用体验的主要途径
    * 注册表（registry）： 是一个巨大的数据库，保存了每个包（package）的信息。
    * 命令行工具 (CLI) ： CLI 通过命令行或终端运行。开发者通过 CLI 与 npm 打交道
